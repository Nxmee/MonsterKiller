Monster Killer
==============

A monster killing game.
--------------

*Alpha Release v0.5*

**Classes**
- Handle functionality of the game
  - Menu 
  - Map Generation
  - Combat

**MonsterKillerSite**
- Website for the game
  - Frontend in regular webstack + stylus
  - Backend in Flask

**States**
- Establish game states
  - Menu
  - Gameplay
  - Combat

**Assets**
- Implements assets of the game
  - Music by luckystar
  - Graphics by Cheez and luckystar
- Often poor spelling
